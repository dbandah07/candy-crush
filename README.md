# Candy Crush (Unity Lab, C#)
Mechanics-focused Unity lab project inspired by match-3 games like *Candy Crush*. Built to practice grid-based logic, swapping, and match resolution.

## Features
- Grid-based board system
- Swap validation and match detection
- Clear and refill loop
- Controls:
  - Desktop: Mouse input for selecting and swapping tiles

## Notes
Completed during Fall 2025 as part of the class **Mobile Game Development**
